# Проект courswork_2(Курсовая 2)
 

## Описание:
Проект courswork_2 - это проект, который получает информацию о вакансиях с платформы hh.ru в России.


##  Функции:
Используемые функции:
filter_vacancies - фильтрация по ключевым словам
get_by_salary - фильтрация по зарплате
sort_vacancies - сортировка по убыванию зарплаты
get_top - Топ N
print_vacancies - Читаемый вывод

## Запуск программы

1. Убедитесь, что установлен Python 3.7+
2. Установите зависимости:pip install -r requirements.txt`
3. Запустите: main.py`
4. Следуйте инструкциям в консоли:
   - Введите поисковый запрос (например: "менеджер") - может долго прогружаться, пожалуйста дождитесь.
   - Укажите количество вакансий для вывода (топ N)
   - При необходимости введите ключевые слова для фильтрации
   - При необходимости укажите диапазон зарплат (формат: min-max)

Программа сохраняет вакансии в файл `data/vacancies.json`
